# Archived docs

This folder holds historical snapshots and low-priority reference material that is not part of the active working doc set.

Archive rules:
- keep docs here when they may still be useful for provenance or occasional reference
- do not treat archived baselines, priorities, or next-step guidance as current truth
- when archive docs disagree with live code or active docs, prefer the active doc stack from `docs/README.md`

Current archive buckets:
- `performance/` — historical audit punchlists
- `research-notes/` — source-reading and reconnaissance notes
